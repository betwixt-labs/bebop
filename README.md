## Intro

Bebop enables schema-based, typesafe, binary serialization and code generation. It is designed to be a good fit for clientâ€“server or distributed web apps that require a faster, more concise, and more typesafe alternative to JSON or MessagePack, while avoiding the complexity of Protocol Buffers, FlatBuffers, and similar solutions.

If you want to get familiar with the schema language and see what the generated code looks like, try out the [web REPL](https://bebop.sh/repl/).
